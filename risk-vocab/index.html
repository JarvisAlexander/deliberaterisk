  • asset: An asset is anything tangible or intangible that is capable of being owned or produced to create value. In this case we are talking about investments. An asset class is a group of assets with similar attributes. Asset classes are not highly correlated with each other and asset with each class can only fit into one class. Finally, the sum of all assets in each class should cover the majority of investable assets and contain a sufficient number of liquid investable assets.
